#StudentManagerSystem
##基于Android的学生信息管理系统
###本项目的内容主要包含三个部分，其中第一部分包括软件启动界面、登录界面、注册界面；其中第二部分包括学生信息列表和用户信息管理菜单界面；其中第三部分包括学生信息编辑界面、学生详细信息显示界面；
